void DBSCAN::run ( const vector< shared_ptr< DataPoint >> &  dataPoints,
vector< int > &  clusterIndexes,
int &  nrOfClusters,
double  eps,
int  minPoints 
)

Run the improved DBSCAN algorithm on the provided set of points.

The implementation of the improved DBSCAN algorithm is based on the paper: T. N. Tran, K. Drab, and M. Daszykowski, Revised DBSCAN algorithm to cluster data with dense adjacent clusters, Chemometrics and Intelligent Laboratory Systems, vol. 120, pp. 9296, Jan. 2013.

Clusters start from index 1, because cluster 0 contains only noise data/points.

Parameters:
dataPointsCollection of data points
clusterIndexesIndexes to which cluster each data point belongs
nrOfClustersTotal number of clusters
epsMaximum distance between two neighbours
minPointsMinimum number of points in one cluster
